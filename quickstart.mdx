## Getting Started

### Invite the Bot to Your Server
1. Use our [official invite link](https://discord.com/oauth2/authorize?client_id=YOUR_BOT_ID&permissions=8&scope=bot%20applications.commands)
2. Select your server from the dropdown
3. Grant the recommended permissions

### Initial Setup
After inviting the bot:
1. Create a "Bleed Bot" role and place it above other roles it needs to manage
2. Use `/setup` to configure default channels and permissions
3. Assign moderator roles with `/modrole set @Role`

## Bot Configuration

<AccordionGroup>
  <Accordion icon="shield-halved" title="Moderation Setup">
    1. Set up your moderation team with `/moderators add @User`
    2. Configure auto-mod with `/automod enable`
    3. Set up logging with `/logchannel #channel`
  </Accordion>

  <Accordion icon="bell" title="Notification Settings">
    - `/welcome enable` - Turn on welcome messages
    - `/welcome message` - Customize the welcome text
    - `/welcome channel #channel` - Set where messages appear
  </Accordion>

  <Accordion icon="sliders" title="Server Preferences">
    - `/prefix set !` - Change command prefix
    - `/language set english` - Set bot language
    - `/timezone set EST` - Configure timezone for timestamps
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Bot isn't responding">
    1. Check if the bot is online in your member list  
    2. Verify you're using the correct prefix (`/` for slash commands)  
    3. Ensure the bot has proper permissions
  </Accordion>

  <Accordion title="Commands not working">
    - Try re-inviting the bot with proper permissions  
    - Use `/sync` to refresh commands  
    - Check our [support server](https://discord.gg/YOUR_INVITE) for outages
  </Accordion>

  <Accordion title="Feature requests">
